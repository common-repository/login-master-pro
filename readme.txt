Get more control over your users and wp-admin

== Description ==

Track who, wherefrom and when logged into wp-admin area. Block users by username or IP address. Limit hourly or daily number of logins per user. Redirect certain user type to a chosen page.

Get more info [here](http://wpsalad.com/login-master-pro/)

== Installation ==

1. Upload `login-master-pro.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
